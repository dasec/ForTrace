Installation
============

This chapter explains how to install fortrace.

.. note::

    This documentation refers to *Host* as the underlying operating systems on
    which you are running fortrace (generally being a GNU/Linux distribution) and
    to *Guest* as the Windows/Linux/Mac OS X virtual machine used to run
    applications in order to create benign network traffic.
