This modul contains the functionallity to realize gui related instructions. Actually only Microsoft Windows virtual machines are supported by this modul.
 
Furthermore this modul will by default connect to the host (fortracemanager - 192.168.100.1:11000) via the network 'local'.
It then receive commands which will be processed. While processing, a substring of the command will be sent to next function, which keyword was hit i.e. 
'application webBrowser open firefox 1 http://www.mydomain.de' which will open a firefox webbrowser (window id 7) and call the website http://www.mydomain.de 
over the next function in  application.py
