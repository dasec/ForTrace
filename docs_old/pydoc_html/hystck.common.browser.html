A browser api to control webbrowsers (host-side).
 
To control the browser the following functions are implemented:
    create
    close
    browseTo
    browseToLinkNumber
    sendKeysToBrowserElement
 
Additionally high level function are supported. For example a login is implemented.
    facebook_login - Login on the main page www.facebook.com  arg: user, pw

class browser
    This class is a remote control on the host-side to control a real browser running on a vm. Supported browsers on the vm
are Mozilla Firefox, Google Chrome. For more details to the correspond vm-side moudle look into fortrace.interactionManager.windows.application.webBrowser (windows).
