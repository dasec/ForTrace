This package is used to create and control virtual machines with the purpose to generate benign network traffic.
 
On the host-side the packages 'core' and 'common' are used.
The package 'core' contains the main class vmHelper which helps to setup virtual machines through the vm class.
In the package 'common' the helper classes and functions are defined. Additionally, application APIs (browser, mailer) are available.
The 'interactionManager' package contain the functions and classes for the vm-side program to interact with the graphical user interface and communicate with the host-side program.
 
The folder examples contains two Python scipts to setup and destroy networks needed by fortrace (setupNetworks/destoryNetworks). Further, the host-side program fortracemanager, as well as the vm-side program interactionmanagerWindows.
